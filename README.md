# Cura QA proof artefacts

Public companion repo holding visual proof (screenshots, A/B previews) for Cura QA pipeline runs.

Each bug investigation produces a folder under `bug-NNN/` with:
- Captured screenshots (light + dark, page header + sheet, before + after where relevant)
- Optional standalone HTML A/B previews

Linked from the canonical PRs on `akshaycj/cura-crm` and from `#cura-watch` Discord notifications.

This repo is **public** so raw image URLs work without auth in PR comments and Discord embeds. It contains no source code — just rendered artefacts.
